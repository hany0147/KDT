# 이차원 리스트
- 리스트를 원소로 가지는 리스트
```python
matrix = [[1,2,3], [4,5,6], [7,8,9]]
matrix = [
  [1,2,3],
  [4,5,6],
  [7,8,9]
]
# 행렬
```
## 특정 값으로 초기화 된 이차원 리스트 만들기
1. 직접 작성
2. for문 활용
3. 리스트 컴프리헨션 활용
    - 반복하면서 append 특정 값을 할 때
    - 어떠한 요소로 구성된 리스트 만들 때
    ```python
    matrix = [[0] * M for _ in range(N)]
    ```

## 입력 받기
1. 행렬의 크기가 미리 주어지는 경우
2. 행렬의 크기가 입력으로 주어지는 경우

## 순회
> 인덱스의 마법사가 되어야 한다.
- len(matrix) -> 행의 길이
- len(matrix[i]) -> 열의 길이

## 전치
- 행과 열을 서로 맞바꾸는 것
