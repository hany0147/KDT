# 딕셔너리(Dictionary)

## 해시 테이블
Non-sequence & Key-Value
- 해시 함수: 임의 길이의 데이터를 고정 길이의 데이터로 매핑하는 함수
- 해시: 해시 함수를 통해 얻어지는 값

### 파이썬의 딕셔너리
- 해시 함수와 해시 테이블을 이용하기 때문에 삽입, 삭제, 수정, 조회 연산의 속도가 리스트보다 빠르다.

## 딕셔너리 메서드
